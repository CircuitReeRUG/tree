API Reference

getLEDCount() - returns LED count

setLEDs([(r,g,b,l), ...]) - set all LEDs (r,g,b: 0-255, l: 0-100)

clearLEDs() - turn off all LEDs

sleep(seconds) - pause (max 10s)

Colors: RED, GREEN, BLUE, YELLOW, CYAN, MAGENTA, WHITE, ORANGE, PURPLE, PINK, OFF

Modules: math, random
